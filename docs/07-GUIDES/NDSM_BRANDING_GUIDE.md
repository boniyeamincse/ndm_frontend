# NDSM Branding Guide

## Canonical Name

- Full name: `Nationalist Democratic Student Movement - NDSM`
- Short form: `NDSM`
- Parent identity reference: `student wing aligned with the Nationalist Democratic Movement (NDM)`

## Motto

- Official motto: `শিক্ষা শৃঙ্খলা প্রযুক্তি সমৃদ্ধি`

## Usage Rules

- Public pages should prefer the full organization name on first mention.
- Navigation, compact cards, and repeated UI can use `NDSM` after initial introduction.
- Avoid inconsistent labels such as `Student Movement - NDM` for the current public brand where the NDSM name is intended.

## Logo Rules

- Use the approved NDSM/organization logo asset consistently.
- Maintain clear space around the logo.
- Do not stretch, recolor arbitrarily, or apply low-contrast placements.

## Color Direction

- Preserve the existing primary brand palette already used in the frontend.
- Gold accent may be used for emphasis, highlights, and ceremonial/official visual cues.
- Bangla motto and official headings should be given strong contrast and prominence.

## Interface Guidance

- Navbar, footer, auth pages, dashboard shell, and public hero areas should use the canonical NDSM naming.
- Public campaign and constitution pages should show the full name where space allows.
- Admin/internal screens may use compact brand labels but should still reference NDSM consistently.
